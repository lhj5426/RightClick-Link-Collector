# RightClick-Link-Collector

为了在不打开网页的情况下右键保存链接而用AI开发的一款

链接保存稍后阅读 网址收藏扩展 

与ONETAB等扩展的区别在于不打开网页可以保存链接

E站很多画廊想下 但是一次打开太多链接 浏览器会很卡

为了控制这个问题

让AI写了一个直接在不打开链接的情况下可以保存链接的功能

这样把想看的画廊的链接先添加到收集器等查看完1000个画廊之后

再回头打开加载下载
