# RightClick-Link-Collector

在不打开网页的情况下右键保存链接,可以导出HTML和TXT

<img width="1474" height="860" alt="image" src="https://github.com/user-attachments/assets/f9b282e7-a366-4da1-a2b5-eebe40704926" />

E站很多画廊想下 但是一次打开太多链接 浏览器会很卡
为了控制这个问题
让AI写了一个直接在不打开链接的情况下可以保存链接的功能
这样把想看的画廊的链接先添加到收集器等查看完1000个画廊之后
再回头打开加载下载
