# 右键链接收集器

一个偏个人工作流的浏览器扩展：在网页上右键保存链接或当前页面，把链接、标题、分组、标签、访问记录和鼠标位置快照一起收进本地管理页。

它适合用来临时收集资料、整理下载页、归档网页入口，或者把一批还没来得及处理的链接先安稳地放进一个地方。

## 功能亮点

- 右键保存链接或当前页面，不需要先打开目标网页。
- 保存时自动记录标题、来源页面、保存时间、分组信息和扩展角标数量。
- 自动截取当前可见页面快照，并记录右键点击位置，后续可在管理页中预览定位。
- 支持自定义分组：名称、背景色、文字色、排序、删除和分组级自动关闭标签页设置。
- 支持标签系统：单条编辑、批量添加、批量删除匹配标签、历史标签快速复用。
- 标签可以是普通文本，也可以是网址标签；网址标签点击后可直接打开。
- 管理页支持全部、缩略图、按分组、按域名、按标签、按日期、未访问等多种视图。
- 支持搜索、重复链接筛选、多关键词高亮、分组跳转和折叠状态记忆。
- 支持批量选择、批量移动、批量删除、批量打标签。
- 支持访问记录统计：点击次数、最后访问时间、未访问聚合视图。
- 支持页数识别和页数排序，对 E-Hentai / ExHentai 画廊链接可自动提取页数与站内标签。
- 支持保存后自动关闭当前标签页，可按全局或分组分别配置。
- 保存成功后在页面内显示提示卡片，可展示分组、总数、快照缩略图和点击位置。
- 支持导出 HTML、TXT，以及完整 JSON 数据备份。
- HTML 导出可选择是否内嵌快照，导出的页面也带搜索、分组、预览和访问标记能力。
- 支持导入备份数据，可选择覆盖或合并。
- 支持浅色、深色、跟随系统主题。
- 支持 Chrome / Edge / Chromium 系浏览器，也提供 Firefox 打包脚本。

## 使用方式

### 保存链接

在网页中的链接上右键，选择“保存链接”相关菜单。  
如果已经创建了分组，右键菜单会显示可选分组；保存后链接会进入对应分组。

### 保存当前页面

在页面空白处右键，选择“保存当前页面”相关菜单。  
这适合把当前标签页本身收进列表。

### 打开管理页

点击扩展图标，或进入扩展选项页，即可打开管理页面。

管理页中可以完成：

- 搜索和筛选链接
- 编辑分组
- 编辑标签
- 预览快照
- 按页数排序
- 查看访问次数和最后访问时间
- 批量操作
- 导出和导入数据

## 安装

### Chrome / Edge / Chromium

1. 打开浏览器扩展管理页。
2. 开启“开发者模式”。
3. 选择“加载已解压的扩展程序”。
4. 选择项目根目录。

项目根目录中使用的是 `manifest.json`。

### Firefox

项目提供了 Firefox 专用 manifest 和打包脚本。

运行：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\build-firefox.ps1
```

脚本会生成：

- `dist/firefox/`
- `dist/right-click-link-collector-firefox.zip`
- `dist/right-click-link-collector-firefox.xpi`

调试时可以在 Firefox 的 `about:debugging` 中临时加载 `dist/firefox/manifest.json`。  
发布或自用分发时可以使用生成的 `.xpi` 或 `.zip`。

## 数据存储

扩展的数据保存在本地浏览器中：

- 链接、分组、标签、设置：`chrome.storage.local`
- 快照图片：IndexedDB，数据库名为 `LinkCollectorDB`

扩展不会把数据上传到远程服务器。  
导出的 JSON 文件可以作为完整备份，建议在重装浏览器或换设备前手动导出一份。

## 导出格式

管理页支持：

- HTML：可选择全部或选中链接，可选择带快照或不带快照。
- TXT：适合快速复制、归档或交给其他工具处理。
- JSON：包含链接、分组和快照数据，用于备份和恢复。

弹出页也提供轻量导出：

- TXT
- CSV
- HTML
- Word `.doc`

## 细节功能

- 快照预览支持大图查看，点击位置会以标记点显示在截图上。
- 详情预览可以在当前结果集中左右切换链接。
- 分组颜色会同步出现在右键菜单保存提示和管理页标签上。
- 搜索支持多关键词，并在页面中高亮命中内容。
- 重复链接可以一键筛出并自动选中，方便批量清理。
- 导出的 HTML 是一个可交互归档页，不只是静态链接列表。
- 快照数据和链接数据分开存储，避免大量图片塞进普通 storage。

## 权限说明

扩展会请求以下权限：

- `contextMenus`：创建右键保存菜单。
- `storage` / `unlimitedStorage`：保存链接、分组、标签、设置和大量快照数据。
- `downloads`：导出 HTML、TXT、JSON 等文件。
- `activeTab` / `scripting`：获取当前页面信息并辅助截图、通知和点击位置记录。
- `<all_urls>`：让右键保存、页面通知和内容脚本能在不同网站中工作。

这些权限主要服务于本地链接采集和管理流程。

## 项目结构

```text
.
├── manifest.json              # Chrome / Edge / Chromium manifest
├── manifest.firefox.json      # Firefox manifest
├── background.js              # 右键菜单、保存逻辑、快照和通知调度
├── content.js                 # 页面内提示、点击位置记录、保存成功通知
├── manager.html               # 管理页
├── manager.css
├── manager.js                 # 管理页交互、分组、标签、导出导入、预览
├── popup.html                 # 扩展弹出页
├── popup.css
├── popup.js
├── db.js                      # IndexedDB 快照存储
├── build-firefox.ps1          # Firefox 打包脚本
└── dist/                      # 构建输出目录
```

## 开发备注

这个扩展目前没有引入构建工具，主要由原生 HTML、CSS 和 JavaScript 组成。  
修改后通常可以直接重新加载浏览器扩展进行测试。

Firefox 包可以通过 `build-firefox.ps1` 重新生成。  
Chrome / Edge 通常直接加载项目根目录即可。

## 适合的场景

- 浏览资料时快速收集链接。
- 下载、资源站、图库页面的批量整理。
- 给链接添加颜色分组和标签。
- 保存“我当时点的是页面哪里”的截图上下文。
- 导出一份可离线查看的 HTML 链接档案。
